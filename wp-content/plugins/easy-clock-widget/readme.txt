== Description ==

This Clock Plugin displays a digital clock as Widget or Shortcode. 

- Local Timezone Display
- 10 Colors to choose from
- Time format 12/24 hours

Color and time format can be set in widget settings.
Shortcode is: [world_clock]

Read more about the Easy Clock Widget at <a href="http://www.worldclockplugin.com/" target="_blank">www.worldclockplugin.com</a>.




== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Appearance-> Widgets and drag the plugin to the desired widget area or paste the shortcode [world_clock] in the desired area.
