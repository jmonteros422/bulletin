This plugin will enable you to use auto expire widget section  


== Description ==
This plugin used to create auto expire text widget according to date.
Once we activate this plugin, it will automatically display a widget in "Available Widgets" section.




== Installation ==
1. Download the file
2. Upload and extract the file the copy the folder into our plugin location like (wp-content/plugins/auto-expire-widget)
3. Activate the plugin from admin side.
4. After activation you can see "Auto Expire Text" widget in "Available Widgets" section 
5. In widget form, please provide the date in (mm/dd/yyyy) format. Otherwise widget will not work.
